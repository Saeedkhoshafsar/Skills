## 🧮 الگوریتم انتخاب (شبه‌کد برای SMART)

```
selected = []
if awareness_pack_incomplete: selected += [01, 06]           # قانون ۱
selected += registry.filter(اولویت=⭐, فاز=فاز_فعلی)
for s in registry.filter(اولویت=➕, فاز=فاز_فعلی, سطح⊇سطح_پروژه):
    if شرط_دقیق(s) با شواهد اسکن تأیید شد: selected += [s]   # نه حدس!
selected -= blacklist                                        # بخش ۳
if len(new(selected)) > 4:
    فعال = ۴ تای بحرانی‌تر؛ بقیه → صف skills-state.md
هر فعال‌سازی → فوراً ثبت در skills-state.md                   # مقاوم به قطعی
اسکیل‌های فاز گذشته → standby
```

## 🔄 اسکیل ناشناخته (در رجیستری نیست)؟

1. SKILL.md اش را بخوان → کاربرد را در یک جمله خلاصه کن
2. با همین ستون‌ها (فاز/سطح/اولویت/شرط) به این فایل اضافه کن
3. سپس مثل بقیه تصمیم بگیر — رجیستری زنده است و با مجموعه‌ی کاربر رشد می‌کند
